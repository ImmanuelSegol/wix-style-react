# `<AddItem/>`

Component for adding new items.

```xml
<AddItem size="small">Add New Item</AddItem>
```

Without action text:

```xml
<AddItem />
```

With tooltip:

```xml
<AddItem size="small" tooltipContent="I am tooltip">
  Add New Item
</AddItem>
```

Theme `image` have fixed icon size and does not allow any action text.
